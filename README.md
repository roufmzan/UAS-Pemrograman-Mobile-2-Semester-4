# UAS-Pemrograman-Mobile-2-Semester-4
## Tugasku - Aplikasi Manajemen Tugas Mahasiswa Terintegrasi AI

Tugasku adalah aplikasi *mobile* berbasis Android yang dirancang untuk membantu mahasiswa mengelola dan mencatat tugas-tugas perkuliahan dengan lebih efisien. Aplikasi ini dikembangkan menggunakan Android Studio dan dilengkapi dengan fitur utama peringkasan *voice-to-text* berbasis AI untuk mempercepat proses pencatatan.

## ✨ Fitur Utama
* **Manajemen Tugas:** Tambah, edit, dan hapus jadwal serta detail tugas perkuliahan.
* **AI Voice-to-Text Summarization:** Mengonversi suara menjadi teks dan meringkas catatan secara otomatis menggunakan integrasi API.
* **Pengingat & Status:** Memantau status tugas (Selesai/Belum) dengan antarmuka yang ramah pengguna.

## 🚀 Cara Menjalankan Aplikasi
1. *Clone* repositori ini: `git clone https://github.com/ldclabs/anda`
2. Buka proyek melalui Android Studio.
3. Sinkronisasi *Gradle* dan pastikan koneksi internet stabil untuk *build* dependensi.
4. Jalankan aplikasi pada emulator atau perangkat fisik (iPhone XS tidak dapat digunakan untuk *running* Android secara langsung, gunakan *device* Android fisik atau emulator pada MacBook Pro Anda).
